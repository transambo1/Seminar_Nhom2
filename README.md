# StormShield

Monorepo for StormShield MVP.

## Architecture

This project is built using a Spring Boot microservice architecture. 
The current running services are:

* **Infrastructure:**
  * `discovery-service` (Eureka Server)
  * `gateway-service` (API Gateway)
  * `mysql-db` (Database layer)

* **Business Microservices:**
  * `auth-service`: User management and security.
  * `alert-service`: **Core Engine**. Automates alert generation based on weather data and manages official emergency broadcasts.
  * `shelter-service`: Resource management for safe havens.
  * `support-service`: Rescue request orchestration.
  * `notification-service`: Real-time event broadcasting.

*Note: The system now prioritizes **Weather-Driven Automation**. Alerts are primarily generated by the `WeatherAlertEngine` within the `alert-service`. Citizen incident reports serve as vital **Field Verification** and situational awareness, supporting the official automated system.*
